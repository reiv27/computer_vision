# Sirius University Course: "Computer Vision"

This repository contains materials and code related to the Sirius University course titled "Computer Vision". In this course, students study the OpenCV library and classical image processing algorithms.

## Course Assignments
- Lab1: MaxMin Filter with register-based acceleration - max_min_filter
- Lab2: Color Filter - color_filter
- Lab3: Object extraction by mask (like portrait mode) - portrait_mode
- Lab4: Marker detection - marker_detection
- Lab5: Moving object detection - moving_object
